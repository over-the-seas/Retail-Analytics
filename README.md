# Scalable Customer Segmentation and Retail Analytics Using Databricks
## Overview
This project analyses retail transaction data using PySpark in Databricks. 
It applies RFM (Recency, Frequency, Monetary) analysis and K-Means clustering 
to segment customers and identify high-value groups.
## Dataset
The dataset used is the Online Retail dataset from the UCI Machine Learning Repository. 
It contains transactional data including customer IDs, product details, quantities, and timestamps.
## Key Analysis
- Revenue by country
- Monthly revenue trends
- Top-selling products
- Customer behaviour analysis
- Hourly sales patterns
- RFM-based customer segmentation
## Machine Learning
K-Means clustering was applied to RFM features to segment customers into distinct groups, 
including low-value, high-value, and VIP customers.
## Technologies
- PySpark
- Databricks
- Pandas
- Matplotlib / Seaborn
- Scikit-learn
## How to Run
1. Open the notebook (.ipynb)
2. Install required libraries
3. Run all cells sequentially
## Key Insights
A small number of high-value customers contribute a large proportion of total revenue, 
while most customers are low-frequency or inactive.
